# Rafistole Fleur · Défi national NDI 2025

## Mission
« Nous ne sommes pas condamnés » : Rafistole Fleur illustre comment un établissement peut réduire sa dépendance numérique tout en gardant le plaisir de créer. L’application immerge l’utilisateur dans un parcours audio-ludique où chaque interaction rappelle les bons réflexes : sobriété logicielle, partage raisonné des ressources cloud et célébration des initiatives locales (fablabs, studios son, clubs green-IT). Le récit embarque l’école dans l’« aventure nerd » responsable : on détourne les écrans pour sensibiliser, pas pour scroller.

## Expérience proposée
- **Narration durable** : déclencheurs textuels indiquent comment couper les flux superflus, recycler le matériel, privilégier l’open-source.
- **Visualisation vivante** : spectre audio 3D qui pulse uniquement lorsque le signal justifie une dépense énergétique, afin de montrer la valeur d’une consommation maîtrisée.
- **Parcours gamifié** : objectifs « anti-sursollicitation » (mode sieste du serveur, badge low-carbon) obtenus en réduisant volontairement l’intensité visuelle.
- **Modes pédagogiques** : bascule micro / musique pour illustrer conférences, ateliers de sensibilisation et ateliers créatifs dans les écoles.
- **Skin Serpentini rétro** : rappelle l’histoire des interfaces low-tech et leur faible empreinte.

## Impacts pour les écoles
1. **Durabilité** : chaque module affiche son coût énergétique estimé et propose une action compensatoire (mutualisation, reconditionné, écoconception).
2. **Anti-dépendance** : sessions limitées, bulles de respiration, incitations à débrancher et passer dans des espaces physiques de co-création.
3. **Responsabilité** : guide intégré pour référents numériques (checklist RGESN, écoconception, inclusion).
4. **Diffusion** : kit prêt à déployer (Vite + static hosting) pour que toute école puisse l’auto-héberger.

## Mise en route locale
```bash
cd src
npm install
npm run dev
```
Puis ouvrir http://localhost:5173/ et accepter les autorisations audio.
