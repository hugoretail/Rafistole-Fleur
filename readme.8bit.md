# readme.8bit

## URL rétro gaming
Hébergé via Rafistole Fleur · route `/?retro=serpentini`

## Instructions
1. Ouvrir l'URL ci-dessus.
2. Attendre l'activation « Serpentini » (icône serpent néon en haut à droite) puis cliquer dessus.
3. Le HUD bascule en pixels 8-bit: scanlines, palette CRT, compteur façon score d'arcade, et serpent chromé qui se balade sur le spectre.
4. Recliquer sur l'icône pour revenir au mode neon classique.

## Ce que cela démontre
- Même pipeline audio mais skinné rétro (gros pixels, fonts bitmap, clignotements CRT).
- Cohérence avec le défi national: même visualisation mais twist nostalgique façon Serpentini.
